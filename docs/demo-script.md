# 60-Second Demo Script

## Main script

### 0:00-0:10

Creators lose margin, distribution, and control the moment they monetize on someone else’s platform. We built a paywall where money moves the instant content unlocks.

### 0:10-0:25

Here’s the creator side. I upload a paid drop, set a price in SOL, choose a transferable access pass, and the app gives me a link plus a QR code I can show anywhere.

### 0:25-0:45

Now the buyer opens that link, connects a wallet, and pays.

Pause while the transaction confirms.

The content unlocks immediately, and the payment is visible on Solana Explorer.

### 0:45-1:00

Because this drop uses an access pass, the buyer can also mint an NFT-backed unlock and carry that access to another wallet. This is not a platform. It is infrastructure.

## If the transaction is slow

- The moment the wallet confirms, the creator still gets paid directly on-chain.
- We keep the Explorer link visible so you can verify the transfer in real time.
- The backup recording uses the exact same route and wallet flow if venue Wi-Fi is noisy.

## If a judge interrupts

We solve the question in the product itself, so let me show the unlock first and then I’ll answer it directly.
