# Live Demo Checklist

## Product checks

- Creator wallet funded.
- Buyer wallet funded.
- One NFT-backed content item already created.
- One QR code ready on the creator screen.
- Buyer route loads directly from a fresh browser tab.

## Network checks

- Wallet extensions installed on both devices.
- Venue or hotspot tested ahead of time.
- Solana Explorer loads quickly.
- Backup recording downloaded locally.

## Slide + script checks

- Deck trimmed to 10 slides or fewer.
- Demo script rehearsed three times.
- One person drives the product.
- One person speaks.
- One person handles interruptions and backup flow.

## If something breaks

- If Supabase is slow: use the pre-created drop.
- If Supabase tables are still missing: create a portable NFT-backed drop and scan that URL instead.
- If payment persistence is slow: show the Explorer confirmation.
- If QR fails: paste the link into the phone directly.
- If live network fails: switch to the backup recording and keep narrating.
